# Strapi Project

This is a sample Strapi project created for demo/testing purposes.

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm run develop
```

Open [http://localhost:1337](http://localhost:1337) in our browser.
